# MLVScanWeb

**MLVScanWeb** is a privacy-first, browser-based malware scanner for Unity mods. It runs [MLVScan.Core](https://github.com/ifBars/MLVScan.Core) entirely in your browser using **WebAssembly** (Blazor).
